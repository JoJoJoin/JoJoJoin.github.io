Godot4—2D碰撞爆炸粒子特效

发表于 2026-04-12 分类于 技术

目标：横向平台跳跃视角，场景中有两个方块，玩家用 WASD 控制其中一个，撞击另一个方块时触发爆炸粒子特效并把目标撞飞，力度越大爆炸越猛烈。

0. 前置知识速览

概念 一句话解释
Node（节点） Godot 里一切东西的基本单位，场景由节点树组成
Scene（场景） 一棵可以保存/复用的节点树，后缀 .tscn
RigidBody2D 受物理引擎驱动的刚体节点，自带速度、重力、碰撞响应
StaticBody2D 不会移动的物理体，用来做地面/墙壁/固定障碍
GPUParticles2D GPU 加速的 2D 粒子发射器，用来做爆炸、火焰等特效
CollisionShape2D 给物理体加上碰撞形状（矩形、圆形等）

2. 配置输入映射（WASD）

先配置好按键，后面测试时才能直接用。

Godot 4 默认的 ui_left/ui_right/ui_up 只绑定了方向键，没有 WASD。手动添加：

  1. 打开 Project → Project Settings → Input Map
  2. 勾选顶部的 Show Built-in Actions（显示内置动作）
  3. 找到 ui_left，点击右边的 + 按钮 → 按下键盘 A → 点击 OK
  4. 找到 ui_right，点击 + → 按下 D → OK
  5. 找到 ui_up，点击 + → 按下 W → OK
  6. 关闭设置窗口

这样 A/D/W 和 ←/→/↑ 都能触发对应动作了。

3. 创建主场景

  1. 菜单 → Scene → New Scene
  2. 选择 Other Node，搜索并选择 Node2D，点击 Create
  3. 重命名根节点为 Main
  4. Ctrl+S 保存为 res://Scenes/Main.tscn

4. 制作地面

  1. 在 Main 下添加 StaticBody2D，重命名为 Ground
  2. 在 Ground 下添加 CollisionShape2D
    • Inspector → Shape → 下拉选 New RectangleShape2D
    • 点击新创建的 Shape，设置 Size = (1200, 40)
  3. 在 Ground 下添加 ColorRect（用来看到地面）
    • Size = (1200, 40)
    • Position 设为 (-600, -20)（让它以 Ground 中心对齐）
    • Color 选一个深灰色，如 #444444
  4. 把 Ground 的 Position 设为 (0, 300) 让它在画面下方

5. 制作目标方块（被撞的那个）

  1. 在 Main 下添加 RigidBody2D，重命名为 Target
  2. 在 Target 下添加 CollisionShape2D
    • Shape → New RectangleShape2D，Size = (64, 64)
  3. 在 Target 下添加 ColorRect
    • Size = (64, 64)，Position = (-32, -32)
    • Color 选红色 #FF4444
  4. 将 Target 的 Position 设为 (300, 248)（站在地面上）

5.1 配置 Target 的 RigidBody2D 属性

选中 Target 节点，在 Inspector 中：

属性 值 说明
Mass 2.0 比 Player 重一点，不会飞太远
Gravity Scale 3.0 和 Player 一致
Lock Rotation ✓ 勾选 防止被撞后疯狂旋转
Linear Damp 2.0 让它滑行后逐渐减速停下（默认 0 = 永远不停）

5.2 给 Target 设置低摩擦材质

  1. Inspector → Physics Material Override → New PhysicsMaterial
  2. Friction 设为 0.05（让它容易被撞滑出去）
  3. Bounce 设为 0.2（略微弹一下更有冲击感）

6. 制作玩家方块

6.1 创建节点

  1. 在 Main 下添加 RigidBody2D，重命名为 Player
  2. 在 Player 下添加 CollisionShape2D
    • Shape → New RectangleShape2D，Size = (64, 64)
  3. 在 Player 下添加 ColorRect
    • Size = (64, 64)，Position = (-32, -32)
    • Color 选蓝色 #4488FF
  4. 将 Player 的 Position 设为 (-200, 248)

6.2 配置 RigidBody2D 属性

选中 Player 节点，在 Inspector 中：

属性 值 说明
Gravity Scale 3.0 让下落感更明显
Lock Rotation ✓ 勾选 防止方块翻滚
Contact Monitor ✓ 勾选 必须开！ 否则无法检测碰撞
Max Contacts Reported 4 至少 ≥ 1，允许报告接触信息
Continuous CD 设为 Cast Ray 防止高速穿透

6.3 设置物理材质（降低摩擦力）

⚠️ 必做！ 默认摩擦系数为 1.0，配合 Gravity Scale 3.0 会产生约 2940 的摩擦力，而脚本只施加 800 的力，方块根本推不动。

  1. 选中 Player 节点，在 Inspector 中找到 Physics Material Override
  2. 下拉选 New PhysicsMaterial
  3. 点击新建的材质展开设置：
    • Friction 设为 0.1（几乎不打滑但也不会卡住）
    • Bounce 设为 0.0（不弹跳，可按喜好调）

⚠️ 重要：Contact Monitor 和 Max Contacts Reported 必须设置，否则 body_entered 信号不会触发！

7. 添加摄像机（跟随玩家）

Player 已在第 6 步创建，现在把 Camera2D 添加为它的子节点。

  1. 选中 Player 节点，点击 +（添加子节点），搜索 Camera2D
  2. 在右侧 Inspector 中勾选 Enabled → On（默认就是开的）
  3. Position 保持 (0, 0)（相对于 Player 的偏移，0 即正中跟随）
  4. Zoom 保持默认 (1, 1) 即可（可稍后微调）

为什么放在 Player 下面？ Godot 中子节点会继承父节点的 Transform，所以 Camera2D 作为 Player 的子节点会自动跟随 Player 移动，无需写任何代码。

8. 制作爆炸粒子特效（3 层）

真实的爆炸效果通常由多层粒子叠加而成。我们一次性创建 3 层：

层级 节点名 作用 视觉效果
第 1 层 ExplosionEffect 主爆炸碎片 黄→橙→红的飞溅碎片
第 2 层 SparkEffect 火花飞溅 小而快、亮白/亮黄的点
第 3 层 SmokeEffect 烟雾扩散 大而慢、半透明灰色的团

8.1 创建主爆炸层（ExplosionEffect）

  1. 在 Main 下添加 GPUParticles2D，重命名为 ExplosionEffect
  2. Inspector 中修改：
属性 值 说明
Emitting false（取消勾选） 不自动发射，代码控制
Amount 64 粒子数量
One Shot ✓ 勾选 只发射一次
Explosiveness 1.0 所有粒子同时喷出
Lifetime 0.6 粒子存活 0.6 秒
  1. Process Material → 下拉选 New ParticleProcessMaterial，点击展开设置：
属性位置 属性 值
Direction Direction (0, 0, 0)
Direction Spread 180（全方向散射）
Initial Velocity Velocity Min 100
Initial Velocity Velocity Max 300
Gravity Gravity (0, 400, 0)（碎片往下落）
Scale Scale Min 2.0
Scale Scale Max 6.0

  1. 设置颜色渐变：找到 Color → Color Ramp → New GradientTexture1D → 点开 → Gradient → New Gradient

    • 左侧色标：#FFD54F（亮黄），Alpha = 255
    • 中间添加色标：#FF6D00（橙色），位置约 0.4
    • 右侧色标：#B71C1C（暗红），Alpha = 0（渐隐消失）

  2. （可选）设置粒子贴图：在 Texture 属性中放一张小的白色方块图（如 4×4 px），不放也行，默认是小方点

8.2 创建火花层（SparkEffect）

  1. 选中 ExplosionEffect，按 Ctrl+D 复制一份
  2. 重命名为 SparkEffect
  3. 修改 Inspector：
属性 值
Amount 32
Lifetime 0.3
  1. Process Material → 点击旁边的 ▼ → Make Unique（必须！否则会影响原来的材质）
  2. 展开材质修改：
属性位置 属性 值
Initial Velocity Velocity Min 300
Initial Velocity Velocity Max 600
Scale Scale Min 1.0
Scale Scale Max 2.0
Gravity Gravity (0, 200, 0)
  1. Color Ramp → Make Unique，修改渐变：
    • 左侧色标：#FFFFFF（纯白），Alpha = 255
    • 右侧色标：#FFD54F（亮黄），Alpha = 0

8.3 创建烟雾层（SmokeEffect）

  1. 选中 ExplosionEffect，Ctrl+D 复制
  2. 重命名为 SmokeEffect
  3. 修改 Inspector：
属性 值
Amount 16
Lifetime 1.2
Explosiveness 0.8（略微错开，更自然）
  1. Process Material → Make Unique，修改：
属性位置 属性 值
Initial Velocity Velocity Min 20
Initial Velocity Velocity Max 80
Scale Scale Min 8.0
Scale Scale Max 16.0
Gravity Gravity (0, -30, 0)（Y 为负，烟雾往上飘）
  1. Color Ramp → Make Unique，修改：
    • 左侧色标：#888888（灰色），Alpha = 150（半透明）
    • 右侧色标：#444444（深灰），Alpha = 0

9. 添加音效节点

  1. 在 Main 下添加 AudioStreamPlayer2D，命名为 HitSound
  2. 准备一个爆炸音效文件（.wav 或 .ogg），拖入 res://Sounds/ 文件夹
  3. 选中 HitSound，在 Inspector 的 Stream 属性中拖入刚才的音效文件

10. 编写玩家脚本

10.1 创建脚本

  1. 选中 Player 节点
  2. 点右上角附加脚本图标（或右键 → Attach Script）
  3. 路径设为 res://Scripts/Player.gd
  4. 语言选 GDScript

10.2 脚本内容

脚本要点说明：

  • last_velocity：碰撞回调时 linear_velocity 已被物理引擎消耗归零，所以每帧提前记录速度
  • amount_ratio：控制粒子发射比例，不直接改 amount（会导致 GPU 缓冲区重建，第二次爆炸可能失效）
  • effect_defaults：在 _ready 中保存每层粒子的初始值，后续用初始值做基数计算，避免多次碰撞后数值越来越小
  • 击飞冲量：impact_speed * 1.5 中的 1.5 是倍率，1.0 = 轻推，3.0 = 弹飞到屏幕外

11. 在编辑器中连接节点引用

  1. 选中 Player 节点
  2. 在 Inspector 右侧找到 Script Variables 部分
  3. 找到 Explosion Effects（数组属性），点击 Array → 设置 Size 为 3
  4. 把场景树中的 ExplosionEffect 拖到 Element 0
  5. 把 SparkEffect 拖到 Element 1
  6. 把 SmokeEffect 拖到 Element 2

12. 运行测试

  1. 按 F6（运行当前场景）或 F5（需要先设置主场景）
  2. 用 A/D 左右移动蓝色方块
  3. 用空格跳跃
  4. 撞向红色方块，观察：
    • 慢速滑过去 → 小爆炸（少量粒子 + 轻推目标）
    • 加速冲过去 → 中等爆炸（目标滑出一段距离）
    • 从高处跳下砸过去 → 大爆炸（满屏粒子 + 目标被撞飞 + 镜头震动 + 音效）

13. 最终场景树一览

常见问题

问题 原因 & 解决
撞了没有粒子效果 检查 Player 的 Contact Monitor = true，Max Contacts Reported ≥ 1
粒子只出现一次，之后不再触发 One Shot 模式下需要调用 restart() 再设 emitting = true
方块穿过地面 地面的 CollisionShape2D 是否正确配置？RectangleShape2D 是否设了尺寸？
A/D 按键没反应 去 Project → Project Settings → Input Map 确认 ui_left/ui_right 绑定了 A/D
玩家方块不停旋转 勾选 Player 的 Lock Rotation
方块推不动 检查是否设了 Physics Material Override，Friction 需要 ≤ 0.1
目标被撞后不滑 检查 Target 是否为 RigidBody2D（不是 StaticBody2D），且有低摩擦材质
第二次爆炸效果变弱 确认脚本中有 effect_defaults 逻辑，用初始值做基数计算

进阶方向

  • 使用贴图：把 ColorRect 替换为 Sprite2D，导入像素画贴图
  • 多个目标：复制多个 Target，修改碰撞检测逻辑判断 body is RigidBody2D 而不是 body.name == "Target"
  • 更多粒子效果：尝试拖尾（Trail）、火焰、闪电等特效
  • 完善操控：加入冲刺、二段跳、dash 等更丰富的操作
